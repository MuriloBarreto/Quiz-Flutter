# Quiz Flutter

O devquiz é um jogo de perguntas para avaliar os seus conhecimentos, desenvolvido na nlw 5

### 🔧 Como Usar

```
   - Clone esse repositório:
   $ git clone https://github.com/MuriloBarreto/Quiz-Flutter.git

   - Entre no diretório:
   $ cd Quiz-Flutter

   - Instale as dependências:
   $ flutter pub get

   - Inicie o app: 
   $ flutter run
  ```
